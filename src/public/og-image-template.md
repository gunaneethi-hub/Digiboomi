# OG Image Design Template for DigiBoomi

## Quick Creation Guide

### Option 1: Canva (Recommended - Free & Easy)

1. **Go to:** [canva.com](https://canva.com)
2. **Click:** "Create a design"
3. **Custom size:** 1200 × 630 pixels
4. **Design elements:**

```
┌─────────────────────────────────────────────────────┐
│  [DigiBoomi Logo]                                   │
│                                                     │
│           Grow Smarter. Grow Faster.                │
│              With DigiBoomi                         │
│                                                     │
│  [SEO Icon]  [Social Icon]  [Email Icon]  [PPC Icon]│
│                                                     │
│  Digital Marketing Agency • Chennai, India          │
└─────────────────────────────────────────────────────┘
```

**Colors:**
- Background: Beige (#F5F1E8)
- Main Text: Black (#000000)
- Accent: Orange (#FF4000)
- Icons: Orange (#FF4000)

**Fonts:**
- Heading: Geist Bold, 72px
- Subtext: Geist Regular, 32px
- Footer: Geist Regular, 24px

**Export:**
- Format: JPG (preferred) or PNG
- Quality: High (100%)
- File name: `og-image.jpg`
- Save to: `/public/og-image.jpg`

---

### Option 2: Figma (Advanced)

1. Create artboard: 1200 × 630px
2. Use same design specs as above
3. Export as JPG (2x quality)
4. Save to `/public/og-image.jpg`

---

### Option 3: Online Tool

**Use:** [ogimage.gallery](https://ogimage.gallery/)
- Select template
- Customize with your branding
- Download as JPG
- Save to `/public/og-image.jpg`
